# SyosetuRatingRank
a web crawler for generating syosetu rating rank list

## Intro
* this script will sort novels from syosetu by **平均評価点 (評価点 / 評価人数) **

## steps
* npm install
* npm run start
* upload the ```result.csv``` to **google spreadsheet** with option ```separator = ;```
